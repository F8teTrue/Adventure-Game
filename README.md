# Adventure Game

**This project is a work in progress of converting from fully text based to a mix of text and graphical elements. Therefore all parts of the game might not yet be fully playable as a graphical and text mixed expirience.**

## 📖 About the Game

**Adventure Game** is a hybrid text/graphical RPG where you explore areas, accept quests, fight monsters, and collect items. Your goal is to complete quests while upgrading your character and surviving the challenges in a mysterious world.

---

## 🎮 Features

### Core Mechanics

- **Exploration**: Traverse unique areas, fight creatures, and discover loot.
- **Quests**: Accept combat or story-driven quests from the Quest Hall to progress.
- **Character Progression**: Earn XP, level up, and improve your stats.
- **Shops**: Buy and sell items like weapons, armor, or potions.
- **NPC Interactions**: Engage with NPCs during story quests to unlock lore and new areas.

### Game Components

| Component         | Description                                                              |
|------------------|--------------------------------------------------------------------------|
| **Character**     | Create and upgrade your adventurer with better gear and attributes.      |
| **Areas**         | Unlock and explore locations with various difficulties and rewards.      |
| **Combat**        | Turn-based battles against a variety of monsters.                        |
| **Story**         | Follow an evolving narrative with branching quests.                      |

## 🚀 Getting Started

### Requirements

- **Python 3.10+**
- OS: Tested on Windows. Should work on macOS/Linux with minor path changes.

### Installation

1. **Clone the repository**

```
git clone https://github.com/F8teTrue/Adventure-Game.git
cd Adventure-Game
```

2. **Install dependencies**

```
pip install -r "requirements.txt"
```

> Alternatively, manually install the required packages (e.g. `colorama`, `keyboard`, `pygame`).

3. **Run the game**

```
python "Adventure Game/main.py"
```

Or run `main.py` from your code editor.

## 📜 How to Play

### Quests

- Visit the **Quest Hall** to accept quests.
- Follow quest objectives like exploring, fighting monsters, or speaking to NPCs.
- Completing story quests unlocks new areas.

### Exploration

- Choose areas to explore based on quest requirements or curiosity.
- Encounter enemies and engage in turn-based combat.
- Find loot such as gear, gold, or potions.

### Shops

- Spend gold earned from exploration or quests.
- Visit the village shop to buy weapons, armor, or potions.

### Navigation

You start at **Home**, which acts as your base. From here, you can:

- **Village** – Visit the shop, Quest Hall, and meet NPCs for story quests.
- **Explore** – Select unlocked areas to travel to and battle creatures.
- **Inventory** – Use or manage items in your possession.
- **Status** – View player stats like level, XP, and equipped gear.
